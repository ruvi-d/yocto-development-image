# Yocto development Docker image

This repo contains Docker file that adds developer tools like [kas](https://kas.readthedocs.io/en/latest/) and [oelint-adv](https://github.com/priv-kweihmann/oelint-adv) to the [crops/yocto](https://hub.docker.com/r/crops/yocto) build image.
For reference on how to use this in a VS Code devcontainer, see [simplest-yocto-setup](https://github.com/ruvi-d/simplest-yocto-setup)
